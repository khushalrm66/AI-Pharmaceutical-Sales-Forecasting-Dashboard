# AI-Powered Pharmaceutical Sales Forecasting Dashboard

## Overview
This project predicts pharmaceutical product demand using Machine Learning and visualizes insights using Streamlit.

## Features
- Sales forecasting
- Revenue analysis
- Inventory optimization
- Trend visualization
- Interactive dashboard

## Tech Stack
- Python
- Pandas
- Scikit-learn
- XGBoost
- Streamlit
- Matplotlib
- Seaborn

## Run Project

### Install Libraries
pip install -r requirements.txt

### Train Model
python train_model.py

### Run Dashboard
streamlit run app.py
